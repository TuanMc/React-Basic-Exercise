##### Task 1.

Create a form to add new contact at the end of in component `Contact.js` (Feel free to play around with styling the form). Use `axios.post()` with the url `https://jsonplaceholder.typicode.com/users` and the body is your form's value to make a request.    render() {

##### Task 2.

Refactor the contact feature to use **Redux** (preferably with `@reduxjs/toolkit`) instead of local component state.

Requirements:

- Create a contacts slice (for example: `contactsSlice.js`) that stores the contact list.
- Add an action/reducer to append a new contact after submit success.
- Dispatch this action from `Contact.js` after `axios.post()` succeeds.
- Render the `<table>` body from Redux state using `useSelector`.
- Use `useDispatch` in the form submit handler.

Suggested shape:

```js
// contactsSlice.js
import { createSlice } from "@reduxjs/toolkit";

const contactsSlice = createSlice({
   name: "contacts",
   initialState: {
      list: [],
   },
   reducers: {
      addContact: (state, action) => {
         state.list.push(action.payload);
      },
   },
});

export const { addContact } = contactsSlice.actions;
export default contactsSlice.reducer;
```

```js
// Contact.js (inside submit handler)
const response = await axios.post("https://jsonplaceholder.typicode.com/users", values);
dispatch(addContact(response.data));
alert("Success!");
```
