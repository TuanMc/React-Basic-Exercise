# Unit Test

<br>

## Exercises

All task files are located in the `react-unit-test-exercise/src/tests` folder. Download the files and write unit tests for each task.

---

##### Task 1 — Test a Simple Utility Function

**File:** `task1/divide.js`

Create unit tests to verify:

- divide() returns correct result
- divide() throws error when dividing by zero


##### Task 2 — Test Array Utilities

**File:** `task2/arrayUtils.js`

Write tests to verify:

- Removes matching item
- Returns original array if item not found
- Works with empty arrays


##### Task 3 — Mock Dependency

**Files:** 
- `task3/helpers/currency.js`
- `task3/services/payment.js`

Mock the currency formatter.

Verify:

- formatCurrency is called
- Correct amount passed
- Correct final string returned


##### Task 4 — Mock API Calls

**File:** `task4/userService.js`

Write tests to verify:

- API success case
- API failure case
- Correct endpoint called
- Correct JSON returned


##### Task 5 — Test React Component

**File:** `task5/Counter.jsx`

Write tests to verify:

- Default count = 0
- Button click increments count
- Multiple clicks work
