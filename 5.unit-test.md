## Exercises

##### Task 1.

Implement **unit tests** for the contact form feature using `Jest` and `React Testing Library`.

Requirements:

- Test that name and email inputs are rendered.
- Test validation errors are shown when submitting empty/invalid values.
- Mock `axios.post()` and test successful submit flow:
   - request is called with correct URL and payload.
   - success alert is shown.
   - `addContact` action updates Redux state (or UI row appears in table).
- Mock failed request and test failed alert is shown.

Suggested test cases:

```js
it("shows validation messages for empty form", async () => {
   // render Contact with Redux Provider
   // click Submit
   // expect validation messages to be visible
});

it("submits form successfully and appends new row", async () => {
   // mock axios.post resolved value
   // fill name/email
   // submit form
   // assert axios called with expected payload
   // assert success alert and new contact row
});

it("shows failed alert when request fails", async () => {
   // mock axios.post rejected value
   // submit valid form
   // assert failed alert is shown
});
```
